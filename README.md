# Vacuum Analysis and High Voltage Conditioning
This repository contains the code developed to analyze pressure data from a photogun vacuum system.

# Features
Code project managment.ipynb : 
- Automated peak detection
- Exponential fitting of pressure decay
- Visual validation of fits

Corrélations.ipynb : 
- Plot of correlations between mesurements

Pressions partielles.ipynb :
- Plot of partial pressures in the photogun

# Usage
Run the main scripts with Python 3 (The scripts won't actually do anything since it needs the data to work).
